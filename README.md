# 🏺 Elegant Interior Work

## Luxury Interior Design Studio | Immersive Digital Experience

**Elegant Interior Work** is a flagship architectural platform built to showcase world-class interior design with an emphasis on "silk-smooth" performance, immersive 360° storytelling, and intelligent client engagement.

---

## ✨ High-End Features

### 🏛️ Immersive Portfolio Architecture

- **360° Panorama Viewer:** A deep-dive experience for luxury spaces, featuring interactive orbit controls and optimized Three.js integration.
- **Before & After Mastery:** High-fidelity interactive comparison sliders for project transformations, synchronized with architectural typography.
- **Intercepting Modals:** Seamless project exploration using Next.js parallel routing for an instantaneous, app-like feeling.

### 🧵 "Silk-Smooth" Interaction DNA

- **Premium Easing System:** A bespoke global animation curve (`cubic-bezier(0.23, 1, 0.32, 1)`) delivering a "weighted" luxury touch across all transitions.
- **Staggered Orchestration:** Coordinated entry animations for client testimonials and the persistent brand sidebar.
- **Mechanical Statistics:** Precision spring-animated counters with architectural physical dynamics.

### 🤖 Intelligent Concierge

- **AI-Powered Chat:** A sophisticated interior design assistant powered by Google Gemini, helping clients explore costs, materials, and project timelines.
- **WhatsApp Integration:** Direct, synchronized bridge to consultants via a persistent high-end floating widget.

## 🚀 Getting Started

### 1. Requirements

Ensure you have Node.js 18+ and a PostgreSQL instance ready.

### 2. Environment Setup

Create a `.env` file based on the provided [`.env.example`](./.env.example):

```bash
# Core AI & Interaction
GOOGLE_GENERATIVE_AI_API_KEY="your-key"
NEXT_PUBLIC_WHATSAPP_PHONE_NUMBER="your-number"

# Data Persistence
DATABASE_URL="your-postgresql-url"
```

### 3. Installation & Database

```bash
npm install
npx prisma generate
```

### 4. Development & Build

```bash
npm run dev    # Launch development server
npm run build  # Create high-performance production bundle
```

The platform standardizes on **Architectural Rhythm**:
